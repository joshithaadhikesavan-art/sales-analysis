# Retail Sales & Profit Analysis Dashboard

##  Overview
This project analyzes retail sales data to generate business insights using Python.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

##  Features
- Sales analysis by category and region
- Top-performing products
- Profit analysis
- Loss-making transactions identification
- Sales vs Profit relationship

##  Key Insights
- Technology category has highest sales
- West region performs best
- Some transactions result in losses
- High sales does not always mean high profit
